# dtool

dtool is a collection of develop tools including:

| Sub command   | Description   | Examples | 
| ------------- | :------------- | ------------ |
| h2s  | Convert hex to string   | $ dtool h2s 0x61626364 <br> abcd  |
| s2h  | Convert string to hex   | $ dtool h2s abcd <br> 0x61626364 |

## Installation
### Via cargo
```
$ cargo install dtool
```

### Via homebrew
```
$ brew install dtool
```
